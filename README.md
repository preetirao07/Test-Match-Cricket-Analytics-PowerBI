# South Africa vs India – Test Match Cricket Analytics
Built an end-to-end Test match cricket analytics dashboard using Power BI, analyzing batting, bowling, and fielding performance


## Project Overview
An end-to-end **Test Match cricket analytics dashboard** built using **Power BI**, focusing on player-level **batting, bowling, and fielding** performance in **South Africa vs India Test matches**.

The data was web-scraped from **ESPNcricinfo StatsGuru**, cleaned and transformed using **Power Query**, and modeled using **DAX** to create interactive and insight-driven dashboards.

## Dashboard Pages
- **Batting Analysis (Test Matches)** – Runs, averages, strike rates, player categories, and rankings
- **Bowling Analysis (Test Matches)** – Wickets, economy, averages, and bowler rankings
- **Fielding Analysis (Test Matches)** – Catches, stumpings, and overall fielding contribution

Each page includes:
- Interactive slicers for player-based filtering
- KPI cards for quick performance insights
- Dynamic ranking visuals for Test match players

## Tools & Technologies
- Power BI
- Power Query (ETL & data transformation)
- DAX (`RANKX`, `POWER`, `ABS`, VLOOKUP-style logic)
- Web Scraping (ESPNcricinfo StatsGuru)
- AI-assisted analytics for metric design and optimization

## Key Outcomes
- Enabled detailed **Test match player performance comparison**
- Built reusable metrics for long-format cricket analytics
- Delivered a clean, interactive dashboard for advanced sports analysis
